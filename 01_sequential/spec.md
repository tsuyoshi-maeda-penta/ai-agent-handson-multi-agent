# AI Agent Design Specification: Weather-Based Fashion Coordinator

## 1. 概要 (Overview)
本ドキュメントは、Antigravity CLI を用いて開発するシーケンシャルパターン（直列処理）のマルチエージェントシステムの仕様を定義します。
このシステムは、ユーザーから指定された地域の「天気予報の取得」を行い、その結果を基に「最適なファッションコーディネートの提案」を直列で実行します。

## 2. システムアーキテクチャ (Architecture)
本システムは、特定の役割を持つ 2 つの AI Agent を直列に繋ぐパイプライン構造を採用します。

```
[User Input] ──> (1. Weather Agent) ──> [Weather Data] ──> (2. Fashion Agent) ──> [Final Proposal]
```

## 3. 技術仕様
* プログラミング言語は Python で実装します。
* Agent Development Kit (ADK) のバージョンは　2.x を使用する。ADK 1.x 系は互換性がないため使用しません。

### ADK のバージョンの指定方法
作成する Python プロジェクトの `pyproject.toml` には、ADK は次のバージョン指定を行います。

```
"google-adk[gcp]>=2.0.0,<3.0.0"
```

## 4. エージェント定義 (Agent Definitions)

### 4.1. Weather Agent (天気予報取得エージェント)
* **Role:** 天気情報アナリスト
* **Goal:** ユーザーが指定したロケーションの正確な現在および直近の天気予報を取得する。
* **Backstory:** 世界中の気象データを分析する専門家。与えられた地域名から、気温、降水確率、天候（晴れ/雨/雪など）の正確なコンテキストを抽出することに長けている。
* **Tools / Functions:** * `get_weather_forecast(location: string)`: 指定された地域の気象データを取得する外部APIツール。Google Maps Platform の Weather API を使用する。

### 4.2. Fashion Agent (コーディネート提案エージェント)
* **Role:** AIパーソナルスタイリスト
* **Goal:** Weather Agent から受け取った気象データに基づき、快適かつファッショナブルな服装を提案する。
* **Backstory:**
  トレンドと実用性を兼ね備えたスタイリングのプロフェッショナル。気温や天候（雨による足元の危険性、直射日光など）を考慮し、ユーザーが快適に過ごせる具体的なコーディネート（トップス、ボトムス、アウター、シューズ、小物など）を提案する。
* **Tools / Functions:** なし（LLMの知識ベースによる推論）

## 5. ワークフロー・タスク定義 (Workflow & Task Definition)

システムは以下の順序でタスクを厳密に実行します（シーケンシャルパターン）。

### Task 1: 天気予報の調査
* **担当エージェント:** Weather Agent
* **インプット:** ユーザーが入力した地域（例: 「東京」「ニューヨーク」）
* **プロセス:** 指定された地域の天気、最高/最低気温、降水確率をツール等を用いて調査する。
* **アウトプット（期待される形式）:** 
```json
  {
    "location": "東京",
    "weather": "雨のち曇り",
    "temperature": { "max": 18, "min": 12 },
    "precipitation_probability": "70%"
  }
```

### Task 2: 天気に合わせたコーディネート提案
* **担当エージェント:** Fashion Agent
* **インプット:** Task 1 のアウトプット（気象データ JSON）
* **プロセス:** 受け取った気温や天候から、適切な防寒性・防水性を持つ衣服を厳選する。また、雨天時の傘の必要性や、気温変化に対応しやすいレイヤード（重ね着）の提案も含める。
* **アウトプット:** ユーザーへの最終提案テキスト

## 6. 出力フォーマット定義 (Output Format)
Fashion Agent が最終的にユーザーに提示する出力は、以下の構成に統一します。

```
🌤️ 【地域名】の本日のコーディネート提案

📊 天気コンディション
* 天候: [Task 1 の天候]
* 気温: 最高 [X]°C / 最低 [Y]°C
* 降水確率: [Z]%

🧥 おすすめのスタイリング
* アウター: 
* トップス:
* ボトムス:
* シューズ:

💡 スタイリングのポイント
* [天候や気温に対して、なぜこの服を選んだのかの理由を記述]
* [傘などの持ち物の推奨]
```
