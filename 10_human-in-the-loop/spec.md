# AI Agent Design Specification: Press Release Agent

## 1. 概要 (Overview)
本ドキュメントは、Antigravity CLI を用いて開発・デプロイを行う、ADK 2.0 ベースの AI エージェントのデザイン仕様書である。

本エージェントは、提供されたソース情報からプレスリリースの初稿を自動作成し、品質レビューを実施した上で、最終的な公開（パブリッシュ）の前に人間の承認を挟む **Human-In-The-Loop (HITL)** パターンを厳格に実装する。これにより、生成AIの自律性と、企業情報発信における人間のガバナンスおよび確実性を両立させる。

### 1.1 目的とユースケース
* **目的**: プレスリリース作成業務の効率化と、誤情報の外部流出防止（ガバナンス強化）。
* **想定ユースケース**: 新製品・新サービスの発表、イベント開催、企業の重要なお知らせにおけるプレスリリース原稿のドラフト作成、校正、および承認ワークフロー。

## 2. システムアーキテクチャ (Architecture)
### 2.1 ワークフロー設計
本システムは、全体を統括する「ルートエージェント」と、専門的なタスクを担う2つの「サブエージェント」による階層型構造で構成される。

### 2.2 処理フロー
1. インプット受信: ルートエージェントがユーザーからプレスリリースの元ネタ（新機能、イベント情報など）を受け取る。
2. 生成・レビュー・ループ (最大3回):
  * プレスリリース作成エージェントが、インプットまたは前回のレビュー結果を基に初稿/改訂稿を執筆する。
  * 品質レビューエージェントが、執筆されたプレスリリースを校正・評価する。
  * 修正点がある場合は「プレスリリース作成エージェント」に戻る（この内部ループは最大3回で打ち切り、強制的に次のステップへ進む）。
3. 人間による確認 (HITL): ルートエージェントが最終成果物をユーザーに提示し、OK または NG (フィードバック付き) の入力を求める。
  * 分岐判定:
    OK の場合: 確定処理を行い、最終成果物を出力してプロセスを終了する。
    NG の場合: ユーザーからの修正指示をコンテキストに含め、ステップ2（生成・レビュー・ループ）へ処理を差し戻す。

## 3. 技術仕様
* プログラミング言語は Python で実装します。
* Agent Development Kit (ADK) のバージョンは　2.x を使用する。ADK 1.x 系は互換性がないため使用しません。

### ADK のバージョンの指定方法
作成する Python プロジェクトの `pyproject.toml` には、ADK は次のバージョン指定を行います。

```
"google-adk[gcp]>=2.0.0,<3.0.0"
```

## 4. エージェント定義 (Agent Definitions)
### 4.1 ルートエージェント (Root Agent)
* 役割: ユーザーとのインターフェース、および全体のワークフロー（HITL）の制御。
* システムプロンプト:
  ```
  あなたはプレスリリース作成プロセスの進行管理を行うマネージャーエージェントです。ユーザーから情報をヒアリングし、サブエージェントに指示を出してください。サブエージェントから上がってきた成果物をユーザーに提示し、承認（OK/NG）を得てください。NGの場合は、ユーザーのフィードバックをサブエージェントに伝えて再生成を指示してください。
  ```

### 4.2 プレスリリース作成エージェント (Sub-Agent 1)
* 役割: 構造化されたインプットおよびフィードバックに基づくプレスリリースの執筆・改訂。
* システムプロンプト:
  ```
  あなたはプロのPRスペシャリストおよびコピーライターです。提供された情報、またはレビューエージェントやユーザーからのフィードバックを基に、メディアの目を引く魅力的なプレスリリースを作成・修正してください。タイトル、リード文、本文、会社概要の標準的な構成を遵守してください。
  ```

### 4.3 品質レビューエージェント (Sub-Agent 2)
* 役割: 作成されたプレスリリースの校正、事実チェック、客観的評価。
* システムプロンプト:
  ```
  あなたは厳格な校正者およびPRコンサルタントです。提出されたプレスリリースのドラフトを、以下の観点からレビューしてください：誤字脱字、トーン＆マナーの適切さ、誇大表現の有無、情報の網羅性。
  修正が必要な場合は具体的な改善案を返し、問題がない場合は「合格」と明記して出力を完了してください。
  ```
