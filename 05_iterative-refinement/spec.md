# AI Agent Design Specification: Ad Copy Refiner

## 1. 概要 (Overview)
本ドキュメントは、Agent Development Kit (ADK) 2.0 および Antigravity CLI を用いて開発する、SNS広告向けキャッチコピー生成エージェントのデザイン仕様書です。
本エージェントは、ユーザーから与えられた商材情報やターゲット層の情報を基に、人間のクリエイターのように高品質なキャッチコピーを自動で創出し、推敲するシステムです。
高品質な出力を保証するため、デザインパターンとして **Iterative Refinement（反復推敲）パターン** を採用し、複数の特化型エージェントの協調動作によって、
一定の品質基準を満たすか最大回数に達するまでキャッチコピーの品質を自律的に向上させます。

## 2. システムアーキテクチャ (Architecture)
### 2.1 ワークフローとデザインパターン
本システムは **Iterative Refinement パターン** をベースとしたループ型アーキテクチャで構成されています。
1回のループは「クリエイティブエージェントによる生成」「レビューエージェントによる評価」「プロンプトリファインメントエージェントによるプロンプト改善」の3ステップからなり、
この一連の推敲プロセスを **最大3回** 反復します。ただし、途中でレビューエージェントが設定された品質基準（スコアなど）を達成したと判断した場合は、
上限回数に達していなくてもループを早期に抜け出し、最終成果物を出力します。

### 2.2 データフロー
1. **インプット:** ユーザーから「商材概要」「ターゲット属性」「配信SNS（Instagram, X, Facebook等）」「訴求軸」が入力される。
2. **ループ処理:** エージェント間で「キャッチコピー案」「評価レポート（スコア・改善点）」「洗練されたプロンプト」のトークン/コンテキストが引き継がれる。
3. **アウトプット:** 最終的に合格基準を満たした、または最大3回推敲された「最高品質のキャッチコピー」および「推敲履歴」を出力する。

## 3. 技術仕様
* プログラミング言語は Python で実装します。
* Agent Development Kit (ADK) のバージョンは　2.x を使用する。ADK 1.x 系は互換性がないため使用しません。

### ADK のバージョンの指定方法
作成する Python プロジェクトの `pyproject.toml` には、ADK は次のバージョン指定を行います。

```
"google-adk[gcp]>=2.0.0,<3.0.0"
```

## 4. エージェント定義 (Agent Definitions)
### 4.1. クリエイティブエージェント (Creative Agent)
* **役割:** 入力された条件および修正されたプロンプトに基づいて、ユーザーの心に刺さるSNS広告用キャッチコピーのバリエーションを生成する。
* **入力:**
  * 初回: ユーザー定義の商材情報、ターゲット、SNSプラットフォームの特性
  * 2回目以降: プロンプトリファインメントエージェントによって最適化された指示文・プロンプト、および前回のレビューフィードバック
* **出力:** キャッチコピー候補（複数案）および各案の意図・解説
* **システムプロンプト（初期値）:**
  ```
  あなたは一流のデジタルマーケティング・コピーライターです。
  与えられた商材情報とターゲット層のインサイトを深く分析し、指定されたSNSに最適化された、クリック率（CTR）を高めるキャッチコピーを生成してください。
  2回目以降の実行時は、提供される「プロンプト修正指示」および「フィードバック」を厳格に反映させ、前回のコピーをさらに魅力的にブラッシュアップしてください。
  ```

### 4.2. レビューエージェント (Review Agent)
* **役割:** 生成されたキャッチコピーを客観的・定量的な基準で評価し、合格か否かを判定する。また、不合格の場合は具体的な改善点を指摘する。
* **入力:** クリエイティブエージェントが生成したキャッチコピー案、および元のユーザー要求
* **出力:**
  * 評価結果（スコア: 1-100）
  * 合格フラグ（例: 85点以上で True、未満で False）
  * フィードバックレポート（ポジティブな点、改善すべき具体的な弱点）
* **システムプロンプト:**
  ```
  あなたは厳格な広告クリエイティブディレクターおよびデータサイエンティストです。
  提出されたキャッチコピー案を、以下の基準で100点満点で採点してください。
    1. ターゲットへの訴求力 (30点)
    2. プラットフォーム適合性（文字数、トーン＆マナー） (30点)
    3. 感情を動かすフック・独自性 (40点)

  総合スコアが85点以上、またはループ上限（3回目）に達した場合は、合格として処理を終了します。
  85点未満の場合は不合格とし、なぜその点数なのか、具体的にどこをどう直すべきか（「もっとベネフィットを具体的に」「専門用語を排除して」など）の詳細なフィードバックを作成してください。
  ```

### 4.3. プロンプトリファインメントエージェント (Prompt Refinement Agent)
* **役割:** レビューエージェントのフィードバックを分析し、クリエイティブエージェントがより良いキャッチコピーを生成できるように、指示（メタプロンプト）を洗練・最適化する。
* **入力:** 直前のループで生成されたキャッチコピー、レビューエージェントからのフィードバック、現在のクリエイティブ用プロンプト
* **出力:** アップデートされたクリエイティブエージェント用の新しいプロンプト指示
* **システムプロンプト:**
  ```
  あなたはプロンプトエンジニアリングの専門家です。
  クリエイティブエージェントが「レビューエージェントからの手厳しいフィードバック」を完全に克服するための、次回の生成用最適化指示（プロンプトの修正・追加分）を構築してください。
  レビューで指摘された弱点（例: ターゲットのペルソナに響いていない、インパクトが薄いなど）をピンポイントで補強するための制約条件や、ペルソナの深掘り指示を既存のプロンプトに動的に注入・ブレンドした、新しいプロンプトを出力してください。
  ```
